KAN neural network & Quantum Machine Learning for ILPD datasets.
We achieved a new SOTA without data leakage, the train/test split follows the original data distribution.

Despite the global burden of liver disease causing two million annual deaths, current ML/QML diagnostics often lack interpretability or suffer from data leakage. 
This paper introduces KQML-Liver, a hybrid framework combining Quantum Machine Learning with Kolmogorov–Arnold Networks (KANs) for enhanced transparency. 
Evaluated on the ILPD dataset with rigorous leakage prevention, the model achieves 82% accuracy and a critical 100% recall, 
successfully detecting all positive cases while outperforming existing baselines.
